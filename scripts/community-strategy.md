# Community Authority Strategy

## Reddit Strategy

### r/MachineLearning
**Posts**: Share original research when published
**Comments**: Help developers with AI agent questions, link when relevant
**Schedule**: 1 post per month maximum

### r/artificial
**Posts**: Discuss AI agent trends in India
**Comments**: Answer questions about tool selection
**No promotional links**

### r/india
**Posts**: India tech adoption topics
**Comments**: Local AI tool discussions
**Value**: Share India-specific insights

### r/programming
**Comments**: Recommend tools for specific use cases
**Links**: Only when directly answering questions

---

## Hacker News Strategy

### Submission Guidelines
- **Topics**: Show HN posts for significant launches
- **Quality**: Only submit when we have substantial news
- **Timing**: Weekdays, 8-10 AM Pacific
- **Title**: Clear, factual, no hype

### Recent Opportunities
- MCP ecosystem report (technical depth)
- India pricing analysis (market insight)
- Open source agent benchmark (data for community)

## Community Rules

### Always
✅ Add genuine value before asking for anything
✅ Share original research and data
✅ Answer questions honestly
✅ Link to specific helpful resources only

### Never
❌ Drop links without context
❌ Promote without disclosure
❌ Spam multiple channels
❌ Misrepresent data
