# Contributing to BestAIAgent.in

Thank you for your interest in improving BestAIAgent.in. This project welcomes accurate pricing updates, tool submissions, link corrections, and transparent community contributions.

## How to Contribute

- Open an issue using the provided issue templates before submitting large changes.
- Keep changes scoped to one logical fix or addition per pull request.
- Follow existing Markdown frontmatter and URL slug conventions.
- Do not add fake claims, ratings, reviews, or hidden promotional content.
- If your change affects a tool, pricing, benchmark, or compliance section, cite the official source.

## Development Setup

```bash
npm install
npm run lint
npm run build
npm run validate:links
npm run validate:canonicals
npm run validate:sitemaps
npm run validate:internal-links
```

## Reporting Issues

Use the issue templates in `.github/ISSUE_TEMPLATE` for tool submissions, pricing corrections, broken links, feature requests, or data corrections.
