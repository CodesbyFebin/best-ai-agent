## Quick Answer
We test AI agents hands-on with the same rigor as a software procurement team. Each tool is scored across 10 dimensions using our proprietary 42-point framework, with special emphasis on India-specific factors like INR pricing, regional language support, and DPDP compliance.

## How We Evaluate

### 1. Hands-On Testing Process
Every AI agent review follows a standardized testing protocol:

| Phase | What We Do | Duration |
|-------|------------|----------|
| **Sandbox Setup** | Install and configure on neutral cloud instances (AWS Mumbai, DigitalOcean Bangalore) | 2-4 hours |
| **Core Workflows** | Execute 15-20 typical use cases for the tool category | 4-8 hours |
| **Benchmark Testing** | Run standardized benchmarks (SWE-bench, GAIA, custom India tests) | 2-4 hours |
| **Integration Tests** | Verify API connections, webhook reliability, and edge cases | 3-6 hours |
| **India Compliance Check** | Audit for GST invoicing, DPDP alignment, regional language support | 1-2 hours |

### 2. Scoring Dimensions (42-Point Framework)

| Dimension | Weight | Evaluation Criteria |
|-----------|--------|-------------------|
| Features | 15% | Capability breadth, unique features, tool integrations |
| Pricing | 15% | Affordability, free tier, INR billing, volume discounts |
| India Fit | 15% | Regional language support, DPDP compliance, local payment methods |
| Security | 10% | SOC2, encryption, access controls, audit trails |
| Compliance | 5% | GDPR, DPDP Act 2023, RBI guidelines, data localization |
| Documentation | 10% | Quality of guides, API docs, examples, India-specific content |
| Performance | 10% | Latency, accuracy, reliability, scalability metrics |
| Community | 5% | GitHub activity, Discord presence, India community engagement |
| Support | 3% | Response time, support channels, India-local support |
| ROI Potential | 2% | Time savings vs. cost for Indian professionals |

### 3. Testing Environments

- **Cloud**: AWS ap-south-1 (Mumbai), GCP asia-south1 (Delhi)
- **Languages**: Tests run in English, Hindi, Hinglish, and regional variants
- **Payment Methods**: INR cards, UPI, net banking where supported
- **Network Conditions**: Simulates typical Indian broadband/mobile speeds

### 4. Review Cadence

- **Monthly Updates**: Scores and pricing checked
- **Quarterly Deep Reviews**: Full hands-on re-testing
- **Annual Audits**: Complete methodology review and updates
