# GPT-4

## Overview

GPT-4 is a large multimodal language model developed by OpenAI, released on March 14, 2023. It is the successor to GPT-3.5 and powers ChatGPT Plus, Microsoft Bing Chat, and other AI assistants.

## Technical Specifications

| Specification | Value |
| -------------- | ----- |
| Parameters | 1.76 trillion |
| Context Window | 8,192 tokens (16K in GPT-4 Turbo) |
| Training Data | Up to September 2021 |
| Architecture | Transformer-based |
| Modality | Text and image input, text output |

## Variants

- **GPT-4**: Original release
- **GPT-4 Turbo**: November 2023, larger context, updated knowledge
- **GPT-4o** (Omni): May 2024, faster, multimodal native

## Performance Benchmarks

| Benchmark | Score |
| --------- | ----- |
| MMLU | 86.4% |
| HumanEval | 73.9% |
| GSM-8K | 86.6% |
| IFEval | 78.2% |

## API Availability

Available through OpenAI API with tiered pricing:
- **GPT-4**: $0.03 / 1K tokens (input), $0.06 / 1K tokens (output)
- **GPT-4 Turbo**: $0.01 / 1K tokens (input), $0.03 / 1K tokens (output)

## Limitations

- **Scientific publications**: Cannot be used for scientific publishing
- **Real-time data**: Knowledge cutoff applies
- **Image recognition**: Limited accuracy for text in images
