## Methodology

Our rankings use a 42-point evaluation methodology across 8 dimensions:

1. **Capability (15%):** Task coverage, model quality, reasoning
2. **Ease of Use (15%):** Setup, UI, docs, onboarding
3. **Features (10%):** Integrations, automations, plugins
4. **Documentation (10%):** API docs, examples, tutorials
5. **Pricing (10%):** Transparency, value, INR estimates
6. **Reliability (15%):** Uptime, latency, error handling
7. **Support (10%):** Response time, channels
8. **Security & Compliance (15%):** Encryption, access, DPDP
