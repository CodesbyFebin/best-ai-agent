## Top 500 Expansion Rules

The market map should scale to 500+ listings without becoming thin or spammy. The rule is simple: a listing can be included when it has a public source and can be mapped to a clear user job.

Accepted source types:

- Official product website
- Official documentation
- GitHub repository or package registry
- Pricing page or changelog
- Public API documentation
- Founder/company announcement
- Independent benchmark or user documentation

Rejected source types:

- Anonymous listicles without sources
- Dead products with no accessible page
- AI wrappers that do not expose a distinct agent workflow
- Claims sourced only from social screenshots
- Tools with no category fit or product identity
