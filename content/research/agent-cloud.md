## FAQ
 
### 1. Which cloud provider offers the best A2A support?
 Google Cloud provides native A2A support through Vertex AI Agents with production readiness in Asia South region.
 
### 2. How do I calculate agent cloud costs in INR?
 Multiply hourly instance cost by usage hours, add token costs in USD converted to INR, plus GST where applicable.
 
### 3. Can I run agent cloud on-premise?
 Yes, Kubernetes-based agent runtimes can be self-hosted with on-premise infrastructure.
 
### 4. What's the difference between agent cloud and traditional cloud?
 Agent cloud includes managed MCP servers, A2A networking, and agent-specific monitoring built-in.
 
### 5. How does data residency work for agent cloud?
 AWS Mumbai, GCP Delhi, and Azure India regions store all data within India borders for compliance.
