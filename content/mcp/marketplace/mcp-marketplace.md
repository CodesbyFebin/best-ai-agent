## Deployment Options

### Quick Deploy (Cloud)
1. Select an MCP server from the marketplace
2. Configure connection settings
3. Deploy with one click
4. Integrate with your AI agent

### Self-Hosted
1. Clone the MCP server repository
2. Configure environment variables
3. Run with Docker or natively
4. Connect via stdio or HTTP

### Docker
```bash
docker run -d \
  --name mcp-github \
  -e GITHUB_TOKEN=your_token \
  ghcr.io/modelcontextprotocol/github-mcp:latest
```

### npm
```bash
npx -y @modelcontextprotocol/server-github
```
