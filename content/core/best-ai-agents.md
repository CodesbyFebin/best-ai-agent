## FAQ

### Getting Started
1. **Where should I start as a beginner?**: Start with Cursor Free (coding) or Dify (building agents). Both are free and have excellent documentation.
2. **Which AI agent is best for Indian students?**: GitHub Copilot (free via GitHub Education) for coding. Dify (free, open-source) for building agents.
3. **How much should I budget for AI agents?**: Free to ₹2,000/month for individuals. ₹10,000-50,000/month for SMBs. Custom pricing for enterprises.

### Pricing & Payments
4. **Can I pay in INR?**: Yes: Yellow.ai, Freshdesk, n8n. No: Cursor, GitHub Copilot, Claude Code (international cards required).
5. **Do Indian students get discounts?**: Yes: GitHub Copilot and Cursor Pro are free via GitHub Education Pack (.edu/.ac.in email).
6. **What GST applies to SaaS subscriptions?**: 18% GST on international SaaS subscriptions. Input tax credit available for GST-registered businesses.

### Compliance & Security
7. **Are AI agents DPDP Act 2023 compliant?**: Self-hosted agents on Indian cloud comply. Verify cloud providers' data handling policies.
8. **Which tools support data residency in India?**: Self-hosted tools on AWS Mumbai or GCP Delhi. Cloud tools: Freshdesk, Zoho, Yellow.ai.
9. **Can AI agents handle Aadhaar data?**: Only with UIDAI-approved APIs and proper encryption. Use self-hosted agents with Aadhaar API integration.

### Technical
10. **Which framework is easiest for Indian beginners?**: CrewAI – simple Python SDK, role-based agent definition, Hindi prompt support.
11. **Can I build voice agents in Hindi?**: Yes: Vapi and Yellow.ai support Hindi TTS. Build custom CrewAI agents with Hindi prompts.
12. **How do I deploy AI agents on AWS Mumbai?**: Use Docker containers with CrewAI, Flowise, or Dify on EC2 or ECS in the ap-south-1 region.

### Business
13. **Which AI agent is best for Indian SMBs?**: Freshdesk (support), Apollo (sales), HubSpot AI (marketing/CRM). All offer free tiers.
14. **Can AI agents integrate with Indian payment gateways?**: Yes, custom agents (CrewAI/Dify) can integrate with Razorpay, Cashfree, and UPI APIs.
15. **What is the total cost of ownership for enterprise AI agents?**: Includes: tool licensing, cloud hosting, integration development, training, compliance. Budget ₹50,000-5,00,000/month.
