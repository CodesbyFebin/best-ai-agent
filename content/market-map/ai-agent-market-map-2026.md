## Category Deep Dive

### 1. General Purpose Agents
**Market Size:** $3.2B | **Growth:** 45% YoY

Leading vendors:
- OpenAI Assistant
- Claude
- Gemini Agents
- Perplexity

**India Signal:** Strong demand for Hindi/Regional language support

### 2. Coding Agents
**Market Size:** $2.1B | **Growth:** 67% YoY

Leading vendors:
- Cursor AI
- GitHub Copilot
- Claude Code
- Windsurf

**India Signal:** Indian developers prefer affordable pricing with GST invoice support

### 3. Business Agents
**Market Size:** $1.8B | **Growth:** 38% YoY

Leading vendors:
- Intercom Fin
- Zendesk AI
- Yellow.ai
- Haptik

**India Signal:** WhatsApp integration is essential

### 4. Voice Agents
**Market Size:** $1.5B | **Growth:** 52% YoY

Leading vendors:
- Vapi AI
- Retell AI
- Bland AI
- ElevenLabs

**India Signal:** Hinglish support and Indian accent training critical

### 5. Framework Agents
**Market Size:** $1.2B | **Growth:** 58% YoY

Leading vendors:
- LangChain
- CrewAI
- LangGraph
- AutoGen

**India Signal:** Local model support (Sarvam, AIcrowd) growing

### 6. MCP Infrastructure
**Market Size:** $890M | **Growth:** 82% YoY

Leading vendors:
- Brave Search
- PostgreSQL MCP
- Redis MCP
- Custom servers

**India Signal:** Indian SaaS connectors in demand
