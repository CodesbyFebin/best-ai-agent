## What Is BestAIAgent.in?

BestAIAgent.in is India's most trusted independent AI agent review platform. We evaluate 50+ AI tools across coding, business automation, voice, and workflow categories with special focus on:

- **INR pricing** with GST calculations
- **Hindi/Hinglish** language support
- **DPDP Act 2023** compliance
- **AWS Mumbai** deployment compatibility
- **WhatsApp Business API** integration

### Core Mission
Build the largest entity graph, comparison database, and benchmark repository for AI agents in India.

## Quick Start

```bash
# Clone repository

# Install dependencies

# Run development server
npm run dev
```

## India-Specific Considerations

### DPDP Act 2023 Compliance
- All self-hosted tools on AWS Mumbai comply
- Vendors must provide DPDP-compliant handling

### GST Invoicing
- 18% GST applies to international SaaS subscriptions
- INR billing available: Yellow.ai, Freshdesk, n8n

### Regional Language Support
- **Hindi**: Yellow.ai, Vapi, Flowise
- **Tamil**: Yellow.ai, Vapi, custom CrewAI
- **Telugu**: Yellow.ai, Vapi
